# 🎮 기능 목록

## 1. 게임 스토리

### 📌 핵심 기능
	•	초기 스토리 텍스트를 화면에 표시한다.
	•	'다음으로' 버튼을 누르면 다음 스토리 페이지로 넘어간다.
	•	마지막 스토리에서 '닫기' 버튼을 누르면 게임 메인 화면으로 돌아간다.


## 2. 게임 로비

### 📌 로비 기능
	•	로비 화면을 표시한다.
	•	플레이어의 현재 시간과 보유 금액을 표시한다.

### 📌 행동 선택
	•	특정 위치에 접근하면 해당 게임을 시작할 수 있다.
	•	특정 위치에 접근하면 부채 관리 프로그램에 접속할 수 있다.

### 📌 부채 프로그램
	•	현재 빚을 확인할 수 있다.
	•	갚을 금액을 입력받고 상환을 처리한다.
	•	남은 부채, 이자율 정보를 확인할 수 있다.

### 📌 예외/검증
	•	상환 금액이 보유 금액보다 많은 경우 오류 메시지를 출력한다.
