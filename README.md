# 🎮 기능 목록

## 1. 게임 스토리

### 📌 핵심 기능
	•	초기 스토리 텍스트를 화면에 표시한다.
	•	'다음으로' 버튼을 누르면 다음 스토리 페이지로 넘어간다.
	•	마지막 스토리에서 '닫기' 버튼을 누르면 게임 메인 화면으로 돌아간다.


## 2. 게임 로비

### 📌 로비 기능
	•	로비 화면을 표시한다.
	•	플레이어의 현재 시간과 보유 금액을 표시한다.

### 📌 행동 선택
	•	특정 위치에 접근하면 해당 게임을 시작할 수 있다.
	•	특정 위치에 접근하면 부채 관리 프로그램에 접속할 수 있다.

### 📌 부채 프로그램
	•	현재 빚을 확인할 수 있다.
	•	갚을 금액을 입력받고 상환을 처리한다.
	•	남은 부채, 이자율 정보를 확인할 수 있다.

### 📌 예외/검증
	•	상환 금액이 보유 금액보다 많은 경우 오류 메시지를 출력한다.


## 3. 경마 게임

### 📌 경주마 정보 제공
	•	경주마 고유의 속도, 스킬, 특징을 제공한다.

### 📌 배팅
	•	선택할 수 있는 경주마들을 배당률과 함께 제공한다.
	•	배팅할 경주마를 선택한다.
	•	배팅할 금액을 선택한다.

### 📌 경마 시작
	•	경주마는 고유의 속도와 무작위의 변동값에 의해 전진한다.
	•	경주마는 그 날의 컨디션에 따라 기본 속도가 달라진다.
	•	경주마는 확률적으로 고유의 스킬을 사용해 빠른 전진을 할 수 있다.

### 📌 경마 종료
	•	도착선에 첫 번째 경주마가 도착하면 게임이 종료된다.
	•	게임 종료 후 일정 시간이 지나면 배당금 지급 씬으로 넘어간다.
	
### 📌 배당금 지급
	•	배팅 결과에 따라 배당금이 지급된다.


## 4. 로또 게임

### 📌 티켓 구매
	•	로또 티켓 가격을 표시한다.
	•	티켓 구매 시 플레이어 자산에서 금액을 차감한다.
	•	잔액이 부족한 경우 구매 불가 메시지를 표시한다.

### 📌 번호 선택
	•	1~20 사이의 숫자 중 4개 번호를 선택할 수 있다.
	•	중복 번호는 선택할 수 없다.
	•	4개 번호를 모두 선택하면 '추첨 시작' 버튼이 활성화된다.
	•	자동 선택 기능으로 랜덤 4개 번호를 선택할 수 있다.

### 📌 추첨
	•	추첨 애니메이션을 재생한다. (번호 5개 → 보너스 번호 순서로 공개)
	•	스킵 버튼으로 애니메이션을 건너뛸 수 있다.
	•	추첨된 5개 번호와 보너스 번호를 화면에 표시한다.

### 📌 결과 판정
	•	플레이어가 선택한 번호와 추첨 번호를 비교한다.
	•	일치한 번호 개수와 보너스 번호 포함 여부에 따라 등수를 결정한다.
	•	등수 정보를 시각적으로 강조해 표시한다.

### 📌 배당금 지급
	•	등수에 따른 배당률으로 상금을 계산한다. (상금 = 구매 금액 × 배당률)
	•	계산된 상금을 즉시 플레이어 자산에 반영한다.
	•	결과 화면에 등수, 상금, 자산 변화량을 표시한다.

### 📌 보조 기능
	•	번호 선택 및 추첨 과정에서 시각/음향 효과를 제공한다.
	•	결과 화면에서 수익률(%)을 표시한다.
	•	결과 확인 후 로비로 이동한다.
