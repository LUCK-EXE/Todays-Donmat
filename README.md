# 🎮 기능 목록

## 1. 게임 스토리

### 📌 핵심 기능
	•	초기 스토리 텍스트를 화면에 표시한다.
	•	'다음으로' 버튼을 누르면 다음 스토리 페이지로 넘어간다.
	•	마지막 스토리에서 '닫기' 버튼을 누르면 게임 메인 화면으로 돌아간다.


## 2. 게임 로비

### 📌 로비 기능
	•	로비 화면을 표시한다.
	•	플레이어의 현재 시간과 보유 금액을 표시한다.

### 📌 행동 선택
	•	특정 위치에 접근하면 해당 게임을 시작할 수 있다.
	•	특정 위치에 접근하면 부채 관리 프로그램에 접속할 수 있다.

### 📌 부채 프로그램
	•	현재 빚을 확인할 수 있다.
	•	갚을 금액을 입력받고 상환을 처리한다.
	•	남은 부채, 이자율 정보를 확인할 수 있다.

### 📌 예외/검증
	•	상환 금액이 보유 금액보다 많은 경우 오류 메시지를 출력한다.


## 3. 경마 게임

### 📌 경주마 정보 제공
	•	경주마 고유의 속도, 스킬, 특징을 제공한다.

### 📌 배팅
	•	선택할 수 있는 경주마들을 배당률과 함께 제공한다.
	•	배팅할 경주마를 선택한다.
	•	배팅할 금액을 선택한다.

### 📌 경마 시작
	•	경주마는 고유의 속도와 무작위의 변동값에 의해 전진한다.
	•	경주마는 확률적으로 고유의 스킬을 사용해 빠른 전진을 할 수 있다.

### 📌 경마 종료
	•	도착선에 첫 번째 경주마가 도착하면 게임 종료 문구가 뜬다.
	•	모든 경주마가 도착하면 다음 씬으로 넘어간다.
	
### 📌 배당금 지급
	•	배팅 결과에 따라 배당금이 지급된다.
